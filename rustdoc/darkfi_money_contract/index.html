Modules

client
Transaction building API for clients interacting with this contract. This module implements the client-side of this contract’s interaction. What we basically do here is implement an API that creates the necessary structures and is able to export them to create a DarkFi Transaction object that can be broadcasted to the network when we want to make a payment with some coins in our wallet. Note that this API doesn’t involve any wallet interaction, but only takes the necessary objects provided by the caller. This is so we can abstract away the wallet interface to client implementations.
