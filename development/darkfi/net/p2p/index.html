P2P provides all core functionality to interact with the peer-to-peer network.

Used to create a network, to start and run it, to broadcast messages across all channels, and to manage the channel store.

The channel store is a hashmap of channel address that we can use to add and remove channels or check whether a channel is already is in the store.
